# Qataar

A token-based queue management system designed for public and private services in Pakistan.

## Features
- Category-based services (Hospitals, Banks, NADRA, DLS)
- Branch-wise token generation
- Service-based queues
- Estimated waiting time calculation

## Technologies Used
- Firebase Firestore
- Git & GitHub
- Flutter / Web 

## Use Case
This system can be implemented in:
- Hospitals
- Banks
- NADRA Offices
- Driving License Centers

## Categories Implemented
- Hospitals
- NADRA
- Banks
- Driving License Centers

## Booking Flow
1. Select category
2. Select branch
3. Select service
4. Generate token
5. View estimated waiting time

## Firestore Structure
categories → category → branches → services
